# ParkingLot

ParkingLot is a parking management system which is non-persistent and managing 
the state inside the GenServer. ParkingLot application provide functionality to
create parking lots, parking slots inside the parking lot.
Once parking slots are crated inside the parking lot we can start parking vehicles.

## API Description

Parking lot provides simple APIs to interact with the application.

- Setup application

```setup
make run
```

- Running test cases

```test
make test
```

ParkingLot will be generated by itself by default 6 slots.

To create new parking lot with the given slot count use the following API

```parking slot

ParkingLot.create_slots(SLOT_COUNT)
```

To park a vehicle in the given parking lot
```Park_vehicle
ParkingLot.park(registration_no, color)
```

Leave_vehicle
```Leave_vehicle
ParkingLot.leave(SLOT_ID)
```

slot_numbers_for_cars_with_colour
```slot_numbers_for_cars_with_colour
ParkingLot.slot_numbers_for_cars_with_colour(COLOR)
```

registration_numbers_for_cars_with_colour
```registration_numbers_for_cars_with_colour
ParkingLot.registration_numbers_for_cars_with_colour(COLOR)
```

slot_number_for_registration_number
```slot_number_for_registration_number
ParkingLot.slot_number_for_registration_number(REGISTRATION_NO)
```
